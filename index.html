Welcome

To start using the app, complete the core setup in this order: add the property owner(s), add properties, set up rental units for each property, define tenancy terms (rent, billing cycle, deposits, start/end rules), add tenants, and then allocate each tenant to the correct unit to activate tenancy management, invoicing, receipts, expenses, inventory, and reporting.
