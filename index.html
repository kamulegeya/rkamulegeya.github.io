The Landlord App is a professional, automation-driven property management platform designed to replace manual spreadsheets and fragmented tools. It helps landlords and agencies manage local and multi-country rental portfolios with accuracy, minimal manual effort, and compliance-ready financial records.

Why it is valuable

  • Rapid Onboarding and Migration at Scale: Bulk import templates support for property owners, rental units, tenants, tenancies, inventory, and expenses
  • Automated billing: invoices can be generated automatically at the start of each billing period based on active tenancies, with review and approval before posting.
  • Fast payment capture: bulk receipt import (Excel/CSV) supports high-volume collections and reduces reconciliation effort.
  • Professional tenant communication: invoices, reminders, receipts, statements, PDFs, and SMS can be produced and shared quickly (where supported).
  • Multi-currency operations: supports multiple currencies across tenancies, invoices, receipts, expenses, and reports, with exchange rate handling where required.
  • Scales with portfolio growth: designed for landlords managing own property(es) ,and agencies managing multiple owners, properties, locations, and currencies.
  • Audit and compliance readiness: preserved history, structured exports, supporting evidence for expenses, and integrity-first workflows support audits and dispute resolution.
  • Automated emailing of documents: invoices,receipts and Payments remaindes are auto sent to tenants.
  • Data security and privacy: data is stored on-device (not on shared servers) and secured using an encrypted SQLite database, supporting offline use and minimizing unauthorized access risk.

What the app covers

  • Tenancy-centric management: tenancies link tenants, units, rent, currency, and billing rules for consistent invoicing and reporting.
  • Invoices and Receipts : one-click PDF invoices/receipts/demand notes and auto-generated tenancy agreements (Word format), where supported.
  • Expenses: expense tracking with evidence attachments (photos/PDF/Word), where supported.
  • Inventory and assets: Inventory items lists, assignments, and disposals to improve handovers and inspections.
  • Reporting and analytics: dashboards for cashflow, arrears, occupancy, payment behavior and trends, with PDF/Excel/CSV exports.
  • Export-first design: structured data exports to support accountants, auditors, and regulators.
  • Rental tax intelligence: country-specific tax profiles (flat, progressive, VAT, withholding, provisional), where configured.
  • Backups and recovery: database backup workflows to protect data and support device migration and operational continuity.
  • Commission computation: commissions can be computed automatically using configurable rules and supporting breakdowns for owner/agent settlement.
  • Late fee computation: late fees can be computed based on defined rules, with transparent calculations for tenant statements and dispute resolution.

Required setup (in order)

To start using the app effectively, complete the core setup in this order: add the property owner(s), add properties, set up rental units under each property, define tenancy terms (rent amount, billing cycle, deposits, start/end rules), add tenants, and then allocate tenants to units to activate tenancy management, billing, receipts, expenses, inventory, and reporting.
